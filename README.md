# Bank Management System

A simple Bank Management System developed in Java using Object-Oriented Programming (OOP) concepts.

## Features

* Create Bank Account
* Display Account Details
* Deposit Money
* Withdraw Money
* Check Balance
* Transaction History
* Interest Calculator

## Technologies Used

* Java
* IntelliJ IDEA
* OOP Concepts

## OOP Concepts Implemented

* Class and Object
* Constructor
* Methods
* Encapsulation
* Conditional Statements
* Loops
* Switch Case

## How to Run

1. Open the project in IntelliJ IDEA.
2. Run the `BankManagement.java` file.
3. Enter account details.
4. Use the menu options to perform banking operations.
